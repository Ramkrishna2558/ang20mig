# GAWS Migration Sandbox

This repository mimics the GAWS layout used for AngularJS to Angular migration practice.

- `gaws02/client` contains the legacy AngularJS-style pages and shared scripts.
- `gaws02/client-ngx` contains the Angular migration target.
- `.codex/skills` contains workflow skills for repeatable migrations and git handling.
- `.agents` contains focused agent role instructions for migration automation.

Use this sandbox to test component-by-component migration instructions before applying them to the production GAWS repository.
